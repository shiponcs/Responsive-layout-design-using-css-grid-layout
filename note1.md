## CSS grid basics
Defintion: CSS grid is a system for structuring layouts using CSS.

Some other ways to structuring layouts was:

1. HTML Tables
2. Box Mode
3. Flexbox
4. CSS Grid

Grid Featires:
- Fixed and Fluid Tracks
- Horizontal and Vertical Alighment
- Naming of Tracks
- Dynamic Organization
